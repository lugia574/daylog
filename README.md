# 🌌 daylog

> 나만의 데스크탑 환경을 완벽하게 만들어주는 커스텀 바탕화면 대시보드 앱입니다.
> 기존 바닐라 자바스크립트 기반의 웹 프로젝트를 **Electron, React, TypeScript**를 활용하여 네이티브 데스크탑 애플리케이션으로 업그레이드했습니다.

## 🚀 프로젝트 소개

단순히 브라우저 새 탭에서 작동하던 기존 Momentum 클론 코딩에서 벗어나, PC 바탕화면에 상주하며 일상과 업무의 효율을 높여주는 **데스크탑 전용 위젯 대시보드**로 재탄생했습니다.

원하는 배경화면 설정은 물론, 일정 관리, 할 일 체크, 실시간 날씨 확인까지 바탕화면에서 한눈에 확인할 수 있습니다.

### 💡 기획 의도 및 업그레이드 포인트

- **웹에서 데스크탑으로:** 브라우저 종속성을 벗어나 Electron을 통해 OS 환경에 자연스럽게 녹아드는 앱 경험 제공.
- **견고한 아키텍처:** React의 컴포넌트 기반 UI 구성과 TypeScript의 정적 타입 검사를 통해 유지보수성과 안정성 향상.
- **세련된 UI/UX:** SCSS를 활용해 디테일하고 유연한 스타일링 적용.

## 🛠 기술 스택 (Tech Stack)

- **Core:** Electron
- **Frontend:** React (Hooks)
- **Language:** TypeScript
- **Styling:** SCSS
- **Build Tool:** Vite (또는 Webpack, 사용하시는 빌드 툴 기입)
- **API:** OpenWeatherMap API (날씨 데이터)

## ✨ 주요 기능 (Features)

- **🖼 커스텀 배경화면 (Custom Wallpapers)**
  - 사용자가 원하는 고화질 이미지를 바탕화면 배경으로 설정.
  - (선택 사항) 시간에 따라 자동으로 배경화면이 변경되는 기능.
- **자유로운 배치**
  - 캘린더 존재 유무, 크기 등 사용자가 기호에 따라 이용할 수 있게 조절할 수 있음.
- **✅ 투두리스트 (To-Do List)**
  - 직관적인 UI로 오늘의 할 일을 추가/수정/삭제/완료 처리.
  - 데이터는 로컬스토리지(또는 파일 시스템)에 안전하게 저장되어 앱을 껐다 켜도 유지됨.
- **📅 캘린더 (Calendar)**
  - 한눈에 들어오는 월간/주간 캘린더 위젯.
  - 주요 일정 마킹 및 확인 기능.
- **🕒 디지털 시계 및 인사말**
  - 현재 시간에 맞는 감성적인 인사말 출력.
- **🕹️ 미니게임**
  - 간단한 미니게임
